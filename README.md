# Datalina
Conversor de graficos en sonidos desarrollado en Pure Data por Damián Anache (2026)

Para su funcionamiento requiere abrirlo en Pure Data con la librería GEM previamente instalada.

Pure Data disponible en : https://msp.ucsd.edu/software.html

Librería GEM instalable vía opción "buscar externals" o desde su sitio: https://gem.iem.at/


Para este proyecto me inspiré en Catalina, el conversor desarrollado por Fernando von Reichenbach y patentado en 1973 (EE. UU.). Pueden encontrar información detallada sobre el trabajo de Fernando von Reichenbach en el Archivo de Música y Arte Sonoro FvR, radicado en la Universidad Nacional de Quilmes: 
https://archivofvr.unq.edu.ar/


También recomiendo esta entrevista, donde comento el trabajo de Fernando y el proyecto Catalina: 
https://www.youtube.com/watch?v=tH-FXIpVwxU


Para el desarrollo de este patch tomé como base el trabajo de Ezequiel Abregú: Code That Sounds: Creative Coding and Sonic Thinking with Pure Data (2025). Capítulo 5: Sonification, sección 5.9 Image Scanner. Disponible en: https://ezequielabregu.github.io/code-that-sounds/


Más información en: http://damiananache.com.ar/datalina
